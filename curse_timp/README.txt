# Десктоп - Электронный журнал

Менеджер оценок, домашней работы и расписания. Три роли - admin, teacher, student, каждой присущ свой функционал.

## Фичи

- Логины/пароли
- Энкрипт с помощью RSA + AES
- Роли:
  - Admin: полный доступ, составление расписания
  - Teacher: управление оценками и домашними заданиями
  - Student: просмотр расписания, оценок и ДЗ

## Системные требования

- Windows 10 (и выше)
- Python 3.8 (и выше)
- Tkinter (преустановлен в Python)
- Пакеты:
  - pycryptodome==3.19.1

## Установка

1. Клонируйте репозиторий/скачайте сорс
2. Убедитесь, что на системе есть Python 3.8
3. В терминале установите нужное:
   ```bash
   pip install -r requirements.txt
   ```

## Первый запуск

1. В терминале запустите приложение:
   ```bash
   python main.py
   ```

2. Приложение само:
   - Создаст нужные директории (`data/` and `data/keys/`)
   - Сгенерирует ключи
   - Создаст дефолтного юзера - admin

3. Данные дефолтного юзера:
   - Username: admin
   - Password: admin

4. После первого лога:
   - Смените пароль (Файл -> Сменить пароль)
   - Вылогиньтесь (Файл -> Выйти из аккаунта)

## Хранение данных

Данные хранятся в зашифрованных .json-файлах в папке `data`:
- `users.json`: информация о юзерах (логины, пароли, роли)
- `subjects.json`: доступные предметы
- `groups.json`: список классов
- `schedule.json`: расписание на неделю
- `grades.json`: оценки
- `homework.json`: ДЗ

## Безопасность

- Пароли хешируются SHA-256
- Файлы шифруются с RSA + AES
- Ключи хранятся в папке `data/keys`

## Проблемы?

1. При ошибке "No module named 'tkinter'":
   - На Windows: tkinter обычно идёт вместе с Python, попробуйте снова вбить команду из "Системные требования"
   - На Linux: установите пакет python3-tk
   - На macOS: установите пакет python-tk

2. При ошибке "No module named 'Crypto'":
   - В терминале: `pip install pycryptodome`
